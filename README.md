# Home Page Layout — CSS Flexbox Practice
 
A starter project for practicing CSS Flexbox. The HTML structure is already built out with a header, hero section, card grid, and footer. Your job is to write the CSS rules in `style.css` and add the correct class names to the HTML elements to bring the layout to life.
 
## Files
- `index.html` — the page structure (don't change this)
- `style.css` — write your CSS rules here
## What you'll practice
`display: flex` · `flex-direction` · `justify-content` · `align-items`
